# Информационная система управления библиотечным фондом

## Описание проекта
Данная система предназначена для автоматизации работы библиотеки. Она включает в себя учет книг, регистрацию читателей, оформление выдачи и возврата книг, контроль сроков возврата, а также формирование отчетности. Цель проекта — упростить и ускорить процессы управления библиотечным фондом.

## Основной функционал
- Ведение каталога книг
- Регистрация и учет читателей
- Оформление выдачи и возврата книг
- Поиск книг по каталогу
- Формирование отчетов о работе библиотеки

## Инструкция по запуску проекта
Поскольку веб-приложение разработано с использованием HTML и CSS и не требует серверной части, запуск осуществляется следующим образом:

1. Скачать или клонировать репозиторий с GitHub.
2. Открыть папку проекта на локальном компьютере.
3. Найти главный файл `index.html`.
4. Открыть файл `index.html` в любом современном браузере (Google Chrome, Microsoft Edge, Mozilla Firefox).

## Публикация проекта через GitHub Pages
Для публикации проекта в интернете можно использовать GitHub Pages:

1. Загрузить проект в репозиторий на GitHub.
2. Перейти в раздел **Settings** репозитория.
3. Открыть вкладку **Pages**.
4. Выбрать ветку `main` и папку `/root`.
5. Сохранить изменения.

После публикации сайт будет доступен по ссылке вида:  
`https://username.github.io/repository-name/`
